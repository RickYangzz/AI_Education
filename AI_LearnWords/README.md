## 创建虚拟环境
- python3 -m venv .venv
- source .venv/bin/activate 
- pip install python-dotenv
- pip install -U pip
- pip install zhipuai
- pip install fastapi_poe
- pip freeze > requirements.txt

## 从git下载下来，启动虚拟环境
- pip install -U pip
- pip install -r requirements.txt
- source .venv/bin/activate

## 关闭虚拟环境
- deactivate

## .env文件内容
- 需要根据api的调用方提供的秘钥进行创建
